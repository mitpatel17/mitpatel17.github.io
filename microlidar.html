Microprocessor Lidar System Project

Stack: Keil Software to program the Microprocessor in C language and Python for data processing and visualization.

This project was part of the Microprocessor Systems Project course which I took in third year where we were tasked to take a set of hardware components including a ToF simple lidar sensor, a stepper motor, and a microprocessor to create a complete embedded system that would map out a given area. The report on the side outlines all the technical spec and the flowcharts to how the system works with images of the results. The hardware components were coded in C using specific hardware API and established I2C and UART communication protocols to communicate the raw distance measurements from the ToF sensor to the connected computer where a Python script would unpack the incoming serial data, process it to get the 3D coordinates and graph it using matplotlib library.

Through this project, I gained knowledge of how embedded systems are created and how they work as well as how to use different serial communication protocols to deliver data between devices and hardware components.
