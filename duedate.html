Due Date Tracker Web App Project

Stack: React.js, CSS, HTML, Node.js, Google Firebase, Bootstrap

CLICK TO SEE GITHUB REPO


This app has user authentication from Google which is handled by Firebase were it will allow different users to sign in, using the user-id to retrieve personal data from the Firebase database which has objects for each due date which may contain different tasks that are due on that day. Date are selected using a calender while other fields like task name are filled into a form. Tasks can be retreated and added through the UI making a easy-to-use productive app.

This was my first project where I individually worked with a cloud-based noSQL database, FireBase, where I learnt its integreation into the web App and how noSQL databases are configuered and designed and how data can be placed and retrieved from one.
