NASA Hackathon Web App Project

Stack: React.js, Java, Node.js, Express, Photoshop

CLICK TO SEE GITHUB REPO


Worked with my parters to create the algorithm that would take the user input which included the data of all the tasks that the astronauts had to complete in the day and how long they take and organize a schedule that would create the best times to sleep during the 12 hours. Also worked in assisting with the UI/UX design of the app with photoshop.

This project was a great experience as it was my first hackathon experience. It taught me how to manage time as there was a time limit to ensure that the project was done on time while maintaining a quality of work. I also got to learn about React as this was my first real react project.
