Big Data Challenge Project

Stack: Python for data processing (Python Pandas, matplotlib), Latex for report generation, Excel

Wrote a publication report for the Undergraduate Big Data Challenge which explained the effects that social media hashtags have on the decision that individuals make regarding getting the Covid vaccine. With the provided data of Twitter Hashtags, I was in charge of use Python (panda library) as well as Excel to analyze the data, generate visual representations, and identify trends in the data in order for my group to link it to their findings and external research. The report on the side represents our findings and how we went about gathering and making sense of the data.

This project taught me how to take big Twitter data, use Excel to organize data with 10000 entries and use Python to effectively filter desired data and to create understandable and revelant graphs that best represents what that data means.
