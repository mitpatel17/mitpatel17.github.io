Pacemaker Project Project

Stack: Python (PySerial, tKinter), Simulink

Given an artificial heart and a pacemaker, we were required to follow the reference documents in order to create a working pacemaker with multiple modes, design and implement the modes in Simulink which would be controlled through a GUI that was created in Python using the tkinter library. Me along with 1 other person in a group of 6 were in charge of creating the Python user interface, implementing user authentication, save user input for each mode, and to be able to show the heart beat from the artificial heart. There was also a lot of debug and unit testing involved for which I wrote python scripts to insure that the parameters of the pacemaker modes were set correctly. The document on the right if the documentation of the whole project that we created, everthing from the Simulink to the Python code and unit testing.

This project taught me a lot about serial communication, the process of software development, and unit testing.
